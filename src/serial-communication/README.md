# 通信方法の規約

## Arduino -> raspberrypi
### フォトリフレクタの値
カンマ区切りで四つ 
```
左, 真ん中左, 真ん中右, 右
```

## raspberrypi -> Ardiuno
### ステッピングモータ
2bit表記
- 00: 直進
- 01: 後進
- 10: 左
- 11: 右

### サーボ
- 0: デフォルトの位置
- 1: 吸った状態

### DCモータ
- 0: 吸わない
- 1: 吸う

```
ステッピングモータ（2bit）サーボ（1bit） DCモータ (1bit)
```
