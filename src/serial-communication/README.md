# 通信方法の規約

## Arduino -> raspberrypi
### フォトリフレクタの値
カンマ区切りで四つ 
```
左, 真ん中左, 真ん中右, 右
```

## raspberrypi -> Ardiuno
### ステッピングモータ
2bit表記
- 000: 止まる
- 001: 直進
- 010: 後退
- 011: 左回り前
- 100: 右回り前
- 101: 左回り後ろ
- 110: 右回り後ろ

### サーボ
- 0: デフォルトの位置
- 1: 吸った状態

### DCモータ
- 0: 吸わない
- 1: 吸う

### 送り方
上が上位ビット, 下が下位ビット
```
0 0 0 DCモータ(1bit) サーボ(1bit) ステッピングモータ(3bit)
```
